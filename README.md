# Shopify Sales Dashboard Analysis

This project showcases a basic sales analysis and visualization of Shopify store data.

## 📊 Project Overview
The goal was to transform raw sales data into actionable insights to understand product performance and profitability.

## 🛠 Tools Used
* **Excel:** For data cleaning, structuring, and calculating profit margins.
* **Visualization:** Dashboard design to highlight key performance indicators (KPIs).

## 📈 Key Findings
* **Total Sales:** 45,390 PKR
* **Profit:** 12,400 PKR
* **Profit Margin:** 27.3%
* **Top Performer:** Cotton Pillows (20,060 PKR)
